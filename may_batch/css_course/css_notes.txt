-> what is css and its use?
    1. css stands for cascading style sheets
    2. used to apply the styles to the html content

the commonly used terminology: property
syntax: propertyName: itsValue;
example: color: red;

-> to apply the styles to the content: 
    - we have 3 different ways
    1. inline styles
    2. embedded styles
    3. external styles
