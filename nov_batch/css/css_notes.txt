-> what is css?
    - the full form of css is cascading style sheets
    - it is used for:
        1. styling of web pages
        2. alignment the contents of web pages

-> property
    syntax: propertyName: itsValue;

-> applying styles to web pages:
    1. inline 
    2. embedded
    3. external

-> commonly used measurements:
    px, in, em, rem, vh, vw, vmin, vmax

-> color:
    1. name: red, 
    2. numerical: rgb(0, 0, 0), rgb(255, 255, 255), rgb(123, 215, 136)
    3. hexadecimal: #a2e

-> custom css:
    1. class 
    2. id
